# final_cloud_22f

Final Project for CS6065: Cloud Computing in the Fall of 2022.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
